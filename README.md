# Country list

## Features:
### - Fetch API data from JSON server
### - Filter data by search field, multiple fields search supported
### - Sort data ASC and DESC by Column
### - Show 20 records per page with Load more

Instructions

```
npm i
```
Terminal 1:
```
json-server --watch db.json --port 3004
```
Terminal 2:
```
npm start
```
